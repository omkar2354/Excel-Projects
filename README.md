Excel Dashboard Projects

This repository contains a collection of interactive Excel dashboards built to analyze business data across different domains. Each project includes a clean dataset, pivot-table analysis, KPIs, and a fully designed dashboard.

📂 Projects Included
1. Ecommerce Sales Analysis Dashboard

Analyzes online sales performance, customer behavior, and revenue trends using interactive charts and KPIs.

2. HR Analytics Dashboard

Provides insights into employee performance, attrition, hiring trends, and workforce distribution.

3. Adidas Sales Analysis Dashboard

Covers product-wise sales performance, regional profitability, and month-wise sales trends for Adidas.

4. McDonald's Sales 2022 Dashboard

Focuses on store-wise sales, monthly growth, customer satisfaction, and overall performance for 2022.
